# SentinelX – Website Security Scanner

A Chrome extension that scans websites for phishing, scams, suspicious domains, unsafe pages, and risky signals.

## Features
- Website trust score
- Phishing signal detection
- HTTPS checks
- Scam / suspicious keyword alerts
- Smart warnings
- Clean popup dashboard
- Scan history
